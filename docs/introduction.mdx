SiafuDB is an embedded, graph-native database that replaces SQLite on the device, at the edge, and inside Web3 pods. Same shape as SQLite — open a file, execute queries, get results — but everything is a graph underneath: relationships are first-class, context is traversable, and data syncs across devices through the Graph Sync Protocol with built-in privacy transformations.

## The Nervous System

SiafuDB, the Graph Sync Protocol, and [NTL](https://ntl.nyuchi.com) form one computing model:

- **AI is the brain.** Reasons, decides, learns.
- **SiafuDB is the memory.** Holds context on every device, at every edge, in every pod.
- **NTL is the nervous system.** Carries signals between nodes through a neural transport layer that learns as it routes.

The Graph Sync Protocol keeps memory coherent across the body. Its training loop lives inside NTL — every sync cycle improves NTL's routing.
