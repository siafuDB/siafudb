## Crate layout

| Crate | Responsibility |
|-------|---------------|
| `siafudb` | Umbrella — re-exports the public API |
| `siafudb-core` | Engine, fragments, schema, mutations |
| `siafudb-sync` | GSP, transforms, conflict resolution |
| `siafudb-access` | Convenience APIs (KV, doc, ts, vector) |
| `siafudb-server` | Optional standalone server — GSPA endpoints, admin |

See [01 — SiafuDB Architecture](/research/01-architecture) for the full crate rationale.

## Versioning

SemVer from 0.1.0 onward. Pre-0.1 the surface is fluid; pin to an exact commit during the alpha.

## Generating docs

```bash
cargo doc --workspace --no-deps --open
```

The Mintlify site hosts the narrative; `cargo doc` is authoritative for signatures.
